# MonteCarloSIPCalculator v1.1.2

V1.1.2 是一次围绕默认体验与结果区可视性的界面打磨版本，重点修正了默认窗口下的纵向可视空间分配，并同步调整了默认参数与图表切换控件样式。

本版本包含：

- 继续压缩左侧控制区与右侧图表区的垂直占用，降低默认窗口下出现滚动条或图表被截断的概率
- 提高最小窗口尺寸约束，避免手动缩放到容易破版的高度
- 更新默认参数为更贴近长期定投情景的一组预设：当前持仓 `0`、每月定投 `1500`、预期年化收益 `15%`、投资期限 `30` 年、极端回撤 `50%`、极端事件平均每 `15` 年 `1` 次、年化波动率 `20%`
- 将三张图表切换按钮统一为圆角矩形风格，和主操作按钮更协调
- 微调“结果概览”标题行与顶部色带的距离，减轻视觉拥挤
- 更新 smoke test，覆盖新的最小尺寸、默认参数和图表区高度约束
- Windows x64 预编译发布包 `MonteCarloSIPCalculator-v1.1.2-win64.zip`

使用方式：
1. 下载 `MonteCarloSIPCalculator-v1.1.2-win64.zip`
2. 解压整个压缩包
3. 运行 `MonteCarloSIPCalculator\\MonteCarloSIPCalculator.exe`

说明：
- 本 Release 为目录型打包结果，`exe` 依赖同目录下的 `_internal/` 文件
- 普通使用场景下不需要额外安装 Python
